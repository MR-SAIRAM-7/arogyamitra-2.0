Quick Setup

Get started with ArogyaMitra in minutes.

1

Clone the Repository

git clone https://github.com/yourusername/arogyamitra.git
cd arogyamitra
2

Setup Backend

cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
python main.py
3

Setup Frontend

cd frontend
npm install
cp .env.example .env
npm run dev
